# nonparamDNAmixtures
Code for non-parametric simulation/replication of DNA mixtures from one-person trace profiles by Kai Budrikas (kaib@itu.dk). This code is part of paper "Non-Parametric Simulation of DNA Mixture Profiles from One-Person Trace Profiles" (2026) by Kai Budrikas and Klaas Slooten. In this code we replicate 2-4-person DNA mixtures from PROVEDIt by pasting together several 1-person DNA profiles, but also simulate some mixtures for a specific set of contributors.

# Necessary software and packages:
- **R** (tested in version 4.4.1)
- **tidyverse**
- **doParallel**
- **openxlsx**

# Explanation of files
The code consists of six R files:
- **00_clean_data.R**: code for cleaning and organizing the data from PROVEDIt.
- **01_replicate_mixtures.R**: code for replicating PROVEDIt mixtures. Outputs raw mixture profiles (without applying stutter filter and analytical thresholds).
- **01_replicate_noisymixtures.R**: code for replicating the same PROVEDIt mixtures, where we take noise from all contributors. Also outputs raw mixture profiles.
- **02_apply_stutterfilter.R**: code for applying analytical thresholds from Riman et al. (2021) and a stutter filter. Also generates collections of trace-POI combinations to go through in LR calculations (by whatever software).
- **03_simulate_mixtures.R**: code for simulating mixtures for one specific case. Simulation parameters are adjustable to whatever case one wishes to simulate mixtures for.
- **helping_functions.R**: collection of functions for transforming profile data from wide to long format and vice versa.

In addition, there are six folders for data:
- **data_provedit**: single-source and mixture profiles from PROVEDIt.
- **data_provedit_cleaned**: single-source and mixture profiles, outputted by 00_clean_data.R.
- **data_replicated**: replicate mixture profiles generated by 01_replicate_mixtures.R and 02_apply_stutterfilter.R, and which are further examined in the paper. "real" represents PROVEDIt data and "mod" represents mixtures pasted from modified 1-person profiles.
- **data_replicated_noisy**: replicate mixture profiles generated by 01_replicate_noisymixtures.R
- **data_simulated**: simulated mixture profiles outputted by 03_simulate_mixtures.R
- **target_genotypes**: random genotypes of contributors from whom we simulated mixtures in 03_simulate_mixtures.R

Results obtained from DNAStatistX can be found from **LR_results**.
